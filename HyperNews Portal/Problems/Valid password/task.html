Valid password Open on JetBrains Academy

Let's write a regexp that checks whether a given string is a valid password. A password can contain lower- and uppercase Latin letters, digits, the underscore _ character, and the hyphen -. It should contain at least 5 characters.

Don't forget that you should check whether the entire string meets the requirements, not only the beginning of the string.

To do that, remember the metacharacters that check for the end of the string.
