Checking inputs Open on JetBrains Academy

Imagine that you want to buy a present for your friend, but you have little money. You want to choose a present which costs more than 120 cents and less than 137 cents.

Implement a function called check() which takes a digit as an argument. The function checks if the digit lies between 120 and 137. If the input is expected, print it. If it is border or invalid, print the message "That's a bad present!"

You don't need to take any input or call the function yourself.
