Imagine that your friend wants to bake a huge apple pie, and he needs a lot of apples. He goes to the market, find a fruiterer and asks for apples. Your friend needs more than 202 apples. Help your friend!

Create a function called check() which takes a string. First, check if the string is a digit. If it isn't, print the message "It is not a digit!" Then, if it is, change the type of your input value to int and check if the digit is more than 202 or equal to 202. It it is, print the digit. Otherwise, print another message "There are less than 202 apples! You cheated on me!"

You don't need to take any input or call the function yourself.
